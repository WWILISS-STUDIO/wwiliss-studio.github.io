# WWILISS.STUDIO website

A static one-page artist website prepared for GitHub Pages.

## Upload to GitHub

1. Open your `wwiliss-studio.github.io` repository.
2. Choose **Add file → Upload files**.
3. Extract the ZIP first.
4. Upload everything inside the `wwiliss-studio-site` folder:
   - `index.html`
   - `css`
   - `js`
   - `assets`
5. Commit the files to the `main` branch.
6. Open `https://wwiliss-studio.github.io`.

## Add real links

Open `js/config.js` and replace each `#` with your real Spotify, YouTube, Apple Music and social profile URLs.

## Important

Keep the folder structure unchanged, otherwise the images and styles will not load.
